# PhD Qualification Exam in Applied Mathematics (Autumn 2025)

---

### 1. (10 points)
Given \(A \in \mathbb{R}^{n \times n}\), consider the sequence \(\{X_k\}\) generated by:
\[ X_{k+1} = 2X_k - X_k A X_k, \quad k = 0, 1, 2, \dots \]
Prove:
(a) If \(\|I - AX_0\| < 1\), then \(A\) is nonsingular and \(X_k \to A^{-1}\) quadratically.
(b) There exists \(X_0\) such that \(\|I - AX_0\| < 1\) if and only if \(A\) is nonsingular.

### 2. (20 points)
Define \(T_0(x) = 1, T_1(x) = x, T_{n+1}(x) = 2xT_n(x) - T_{n-1}(x)\).

(a) Show that \(T_n(x) = \cos(n \arccos x)\) for \(|x| \leq 1\) and \(\cosh(n \text{arccosh } x)\) for \(|x| > 1\).
(b) Prove that \(\min_{\text{deg } p \leq n, p(1)=1} \|p\|_\infty = 1\), attained at \(p = T_n\).
(c) For \(f \in C^{n+1}[-1, 1]\), with nodes as zeros of \(T_{n+1}(x)\), prove:
\[ \|f - L_n\|_\infty \leq \frac{\|f^{(n+1)}\|_\infty}{(n+1)! 2^n}. \]
(d) Derive the Gauss quadrature formula for \(I(f) = \int_{-1}^1 \frac{f(x)}{\sqrt{1-x^2}} dx\) and prove:
\[ |I(f) - I_{n+1}(f)| \leq \frac{\pi \|f^{(2n+2)}\|_\infty}{(2n+2)! 2^{2n+1}}. \]

### 3. (20 points)
Generalized eigenvalue problem: find \(x \neq 0, \mu, \lambda \in \mathbb{C}\) s.t. \(|\mu|^2 + |\lambda|^2 = 1, \mu Ax = \lambda Bx\).

(a) Give an example where \((A, B)\) has more than \(n\) eigenvalues.
(b) Prove there exists a UET s.t. \(Q^H(A, B)Z = (S, T)\) with \(S, T\) upper triangular.
(c) How to obtain eigenvalues from \((S, T)\)?
(d) Construct a UET s.t. \(A_0\) is upper Hessenberg and \(B_0\) is upper triangular (assume \(n=4\)).

---

### 4. (15 points)
Construct the Du Fort-Frankel scheme for the 2D diffusion equation \(u_t = u_{xx} + u_{yy}\) and discuss its consistency and stability.

---

### 5. (15 points)
For \(u_{tt} = u_{xx}\), analyze the stability of:
\[ \frac{u_j^{n+1} - 2u_j^n + u_j^{n-1}}{\tau^2} = \frac{\delta_x^2 u_j^{n+1}}{4h^2} + \frac{\delta_x^2 u_j^n}{2h^2} + \frac{\delta_x^2 u_j^{n-1}}{4h^2}, \]
where \(\delta_x^2 u_j = u_{j+1} - 2u_j + u_{j-1}\).

### 6. (20 points)
Rayleigh oscillator: \(y'' + y + \epsilon[\frac{1}{3}(y')^3 - y'] = 0, y(0)=0, y'(0)=2a\).
(a) For small \(\epsilon\), construct an asymptotic solution valid for large \(t\).
(b) State the accuracy and explain.
(c) Plot orbits in phase plane and explain observations.

---

### 7. (20 points)
Minimization: \(\min_{x \in \mathbb{R}^n} \|Ax - b\|_2 + \gamma\|x\|_1\).
(a) Derive the Lagrange dual of the equivalent problem: \(\min \|y\|_2 + \gamma\|x\|_1\) s.t. \(Ax - b = y\).
(b) Let \(r = \frac{Ax^* - b}{\|Ax^* - b\|_2}\). Show \(\|A^T r\|_\infty \leq \gamma\) and \(r^T Ax^* + \gamma\|x^*\|_1 = 0\).
(c) Show if \(\|a_i\|_2 < \gamma\), then \(x_i^* = 0\).
